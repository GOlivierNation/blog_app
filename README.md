## Getting Started
- clone the repository by running\
    `https://github.com/GOlivierNation/blog_app.git`
- navigate to the folder\
    `cd blog_app`
- Install packages\
    `bundle install`
- Setup database\
    `rails db:setup`
- Run migration\
    `rails db:migrate`
